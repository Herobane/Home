# Home #
### Plugin Minecraft pour Bukkit 1.8.8
##### Dernière version stable : 1.2

Home est plugin Minecraft qui permet de définir des endroits spécifiques (Maisons, spawns, warps) et de s'y téléporter en une commande

## Commandes : #
* /sethome
* /home
* /setwarp <nom_du_warp>
* /warp <nom_du_warp>
* /setspawn
* /spawn

## Description des commandes : #
* /sethome  : permet de définir la maison du joueur sur le block où il se trouve
* /home : permet au joueur de se téléporter à sa maison à condition que celle-si soit définie
* /setwarp : permet au joueur de définir un warp commun à tout les joueurs
* /warp : permet au joueur de se téléporter au warp précisé en paramètre
* /setspawn : permet de définir un spawn publique à tout les joueurs
* /spawn : permet de se téléporter au spawn

###### Historique des versions : ######
* v1.1 : Ajout de la gestion des warps (/setwarp et /warp)
* v1.2 : Ajout de la gestion du spawn (/setspawn et /spawn) + Gestion de la compatibilté entre plusieurs maps
